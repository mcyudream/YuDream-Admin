# Agent 工作流完整运行时实施计划

1. 为图执行上下文、拓扑校验、条件分支和变量解析编写失败测试，实现独立 `AgentWorkflowExecutor`。
2. 为模板、条件、代码、引用节点编写失败测试并实现处理器。
3. 为大模型、问题理解、多模型连续执行编写失败测试并接入 AI 网关。
4. 为搜索、向量、重排、Embedding 编写失败测试并接入 Wiki 网关。
5. 增加文档提取网关与 Apache Tika 适配器，覆盖文本、Office、PDF 输入。
6. 将 `AgentAppService` 调试与正式运行统一委托给工作流执行器，保留工具授权和 SSE 事件。
7. 增加安全元数据接口，补全所有节点检查器配置、类型和请求映射。
8. 运行后端测试/编译、前端类型/规范测试，并使用浏览器验证实际分支、多节点和滚动布局。
